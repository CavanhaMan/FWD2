Testando Bootstrap com CDN!

Nesta prática, instalamos os arquivso via CDN!

CDN é a sigla para Content Delivery Network ou seja, “Rede de Fornecimento de Conteúdo”. É um grupo de servidores espalhados em muitos locais. Esses servidores armazenam cópias duplicadas de dados para que os servidores possam atender às solicitações de dados com base nos quais servidores estão mais próximos dos respectivos usuários finais.
