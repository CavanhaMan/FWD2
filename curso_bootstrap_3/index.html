Testando Bootstrap com NPM!

Nesta prática, instalamos os arquivso de bootstrap via NPM!

npm (originalmente abreviação de Node Package Manager)[3] é um gerenciador de pacotes para o Node.JS.

npm é o gerenciador de pacotes padrão para o ambiente de tempo de execução JavaScript Node.js. Ele consiste em um cliente de linha de comando, também chamado de npm, e um banco de dados online de pacotes públicos e privados pagos, chamado de registro npm. O registro é acessado por meio do cliente e os pacotes disponíveis podem ser navegados e pesquisados no site do npm. O gerenciador de pacotes e o registro são gerenciados pela npm.

O NPM permite, para além da criação de módulos, executar instruções ou conjuntos de instruções através de um comando criado pelo utilizador conforme a sua necessidade.
